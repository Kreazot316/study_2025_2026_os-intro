Также были использованы различные опции команды \texttt{ls} для
получения более подробной информации о содержимом каталога
\texttt{/tmp}:

\begin{verbatim}
ls -a
ls -l
ls -F
ls -alF
\end{verbatim}

Обычная команда \texttt{ls} показывает только имена файлов и каталогов.
Опция \texttt{-a} добавляет скрытые файлы и каталоги, имена которых
начинаются с точки. Опция \texttt{-l} выводит подробную информацию:
права доступа, число ссылок, владельца, группу, размер и дату изменения.
Опция \texttt{-F} добавляет символ, указывающий тип объекта: \texttt{/}
для каталога, \texttt{*} для исполняемого файла, \texttt{@} для
символической ссылки. Комбинированный вариант \texttt{ls\ -alF}
объединяет все эти возможности.
\end{block}

\begin{block}{4.3 Просмотр описания основных команд}
\protect\phantomsection\label{ux43fux440ux43eux441ux43cux43eux442ux440-ux43eux43fux438ux441ux430ux43dux438ux44f-ux43eux441ux43dux43eux432ux43dux44bux445-ux43aux43eux43cux430ux43dux434}
Использовал команду \texttt{man} для просмотра описания следующих
команд: \texttt{cd}, \texttt{pwd}, \texttt{mkdir}, \texttt{rmdir},
\texttt{rm}. (рис.~\ref{fig-014}).

\begin{figure}

\centering{

\includegraphics[width=0.7\linewidth,height=\textheight,keepaspectratio]{image/14.png}

}

\caption{\label{fig-014}Просмотр описания команды cd}

\end{figure}%

\begin{itemize}[<+->]
\tightlist
\item
  \texttt{cd} --- переход в другой каталог. Без аргументов переходит в
  домашний каталог, \texttt{cd\ ..} поднимает на уровень выше.
\item
  \texttt{pwd} --- выводит абсолютный путь текущего каталога.
\item
  \texttt{mkdir} --- создаёт каталоги. Опция \texttt{-p} позволяет
  создавать промежуточные каталоги.
\item
  \texttt{rmdir} --- удаляет только пустые каталоги.
\item
  \texttt{rm} --- удаляет файлы. Для удаления каталогов используется
  \texttt{-r}, а для подтверждения удаления --- \texttt{-i}.
\end{itemize}
\end{block}
